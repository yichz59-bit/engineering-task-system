# engineering-task-system

## 项目结构

此仓库为前后端分离工程模板，前端已初始化 Vue 3 + TypeScript + Vite 项目。

- `frontend/`：前端代码目录
  - `src/views/`：页面视图
  - `src/types/`：前端 TypeScript 类型定义
  - `src/mock/`：前端静态 mock 数据，例如任务管理页演示数据
- `backend/`：后端代码目录
- `docs/`：项目文档或设计说明
- `scripts/`：自动化脚本和开发辅助脚本
